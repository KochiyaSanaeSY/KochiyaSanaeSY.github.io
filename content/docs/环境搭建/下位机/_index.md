# 下位机环境搭建

## 1. 克隆 ArduPilot 代码库

在 Ubuntu 22.04 中打开一个终端，输入如下命令：

`git clone --recurse-submodules https://github.com/ArduPilot/ardupilot.git`

下载完成之后，我们会在目录下找到 “ardupilot” 这个文件夹。
